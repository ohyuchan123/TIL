# 1. 서버(Server)란?

## 서버
서버도 그냥 컴퓨터이다. 서버라는 역할을 하면 서버 컴퓨터로 불리는 것이다.
즉, 우리가 지금 사용하고 있는 컴퓨터,노트북도 서버 컴퓨터가 될 수 있는 것이다!

서버란 역할을 하면 서버컴퓨터라고 불린다고 했다. 그럼 서버라는 역할은 무엇일까?

## 서버의 역할
`서버`는 컴퓨터 네트워크에서 중앙 집중적으로 관리되는 컴퓨터 시스템으로, 클라이언트 컴퓨터에서 요청한 데이터나 서비스를 처리하고 제공하는 역할을 합니다.

이를 통해 다수의 클라이언트 컴퓨터가 서로 연결된 네트워크에서 데이터를 공유하고, 정보를 주고받을 수 있습니다. 서버는 다양한 종류의 서비스를 제공할 수 있습니다.

예를 들면 웹 서버,이메일 서버,파일 서버 등이 있습니다. 또한 서버는 데이터
저장 및 관리, 보안,백업 등과 같은 기능을 수행하여 네트워크의 안정성과 보안성을 유지합니다.
